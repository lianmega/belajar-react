### Soal 1
Pada gambar tersebut, silakan Anda browsing apa yang dimaksud dengan:

 1. TypeScript
    Bahasa pemrograman open-source yang dikembangkan oleh Microsoft. Ini adalah superset dari JavaScript, yang berarti semua kode JavaScript valid juga merupakan kode TypeScript, tetapi TypeScript menambahkan fitur-fitur tambahan seperti pengetikan statis, antarmuka, enumerasi, dll. TypeScript memberikan kemampuan untuk menulis kode JavaScript yang lebih terstruktur dan mudah dipelihara, terutama untuk proyek-proyek besar.

2. ESLint
    Utilitas linting untuk JavaScript. Ini digunakan untuk menganalisis kode JavaScript Anda dan memberikan peringatan tentang pola-pola yang tidak diinginkan atau potensi bug dalam kode Anda. ESLint dapat dikonfigurasi dengan berbagai aturan yang dapat disesuaikan, sehingga Anda dapat menyesuaikannya sesuai dengan gaya kode dan preferensi Anda sendiri.

3. Tailwind CSS
    Framework CSS yang unik, yang berbeda dengan framework CSS lainnya seperti Bootstrap atau Foundation. Alih-alih menggunakan kelas-kelas CSS yang telah ditentukan sebelumnya, Tailwind memungkinkan Anda untuk membangun desain web dengan menentukan gaya langsung dalam HTML menggunakan kelas-kelas kecil yang dapat disesuaikan. Ini memberikan fleksibilitas yang besar dalam desain, sambil mempertahankan struktur dan konsistensi yang baik.

4. App Router
    Merujuk pada perpustakaan atau framework yang digunakan untuk menangani rute (routing) dalam aplikasi web atau aplikasi berbasis web. Ini memungkinkan Anda untuk mengatur bagaimana aplikasi Anda menanggapi URL yang diakses oleh pengguna, dan memuat komponen atau halaman yang sesuai dengan URL tersebut. Contoh-contoh populer dari App Router adalah React Router untuk aplikasi React dan Vue Router untuk aplikasi Vue.js.

5. Import alias
    Cara untuk memberikan nama yang lebih singkat atau lebih deskriptif untuk modul atau berkas yang diimpor dalam JavaScript atau TypeScript. Ini memungkinkan Anda untuk merujuk ke modul tersebut dengan nama yang lebih mudah diingat dan lebih mudah ditulis. 

### Soal 2

Berikut adalah penjelasan tentang kegunaan folder dan file pada struktur proyek yang Anda tunjukkan:

1. .git: Folder ini digunakan oleh Git untuk kontrol versi. Folder ini menyimpan metadata dan database objek untuk proyek.
2. node_modules: Berisi semua paket dan modul yang diinstal melalui npm, termasuk dependensinya.
3. public: Biasanya berisi file statis yang dapat diakses secara publik, seperti gambar, stylesheet, atau script.
4. src: Folder sumber di mana kode utama aplikasi berada. Biasanya berisi komponen, aset, dan file kode lainnya.
5. .eslintrc.json: File konfigurasi untuk ESLint, alat untuk mengidentifikasi dan memperbaiki masalah dalam kode JavaScript.
6. .gitignore: Mendaftar file dan direktori yang diabaikan oleh Git; mereka tidak akan dilacak atau ditambahkan ke sistem kontrol versi.
7. next.config.mjs & next-env.d.ts: File konfigurasi terkait pengaturan kerangka kerja Next.js dan tipe lingkungan masing-masing.
8. package.json & package-lock.json: Berisi metadata tentang proyek dan dependensi; package-lock memastikan versi dependensi tetap konsisten di semua lingkungan.
9. postcss.config.js: File konfigurasi untuk alat pemrosesan PostCSS yang digunakan dalam mengubah gaya CSS dengan plugin JavaScript.
10. README.md: File markdown yang berisi informasi tentang proyek seperti deskripsi, instruksi pengaturan, dll.
11. tailwind.config.ts: File konfigurasi Tailwind CSS untuk menyesuaikan berbagai aspek konfigurasi default Tailwind termasuk warna, spasi, dll.
12. tsconfig.json: Konfigurasi untuk kompiler TypeScript dengan pengaturan yang mendefinisikan bagaimana TypeScript harus dikonversi menjadi JavaScript.
